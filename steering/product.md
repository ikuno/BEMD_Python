# BEMD-Python - Product Context

## 概要

BEMD-Python は、Bidimensional Empirical Mode Decomposition（二次元経験的モード分解）の Python 実装です。
MATLAB 版 `bemd.m` を numpy / scipy ベースで忠実に移植しています。

## 目的

2D 信号（画像）をデータ駆動的に複数の IMF（固有モード関数）と残差に分解する。
フーリエ変換と異なり、非線形・非定常な信号にも適用可能な手法を提供します。

## ステークホルダー

- 信号処理・画像解析の研究者
- BEMD を利用するアプリケーション開発者
